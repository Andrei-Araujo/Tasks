# APP de Tasks

Para instalar:

```
git clone https://github.com/Andrei-Araujo/Tasks.git
cd Tasks
npm i
```

Para executar:
  - Executar servidor
    1) ```cd api```
    2) ```npx nodemon```
  
  - Executar client
    1) ```cd site```
    2) ```npm start```
